We can use a sign chart for the second derivative to find inflection points in a similar way that we used a sign chart for the first derivative to find relative extreme values. We find places where the second derivative is zero, put that value on a number line, and check the sign of the second derivative on either side. If the sign changes (that is, if concavity changes) from one side to the other, then we have an inflection point.
