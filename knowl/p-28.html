Remember that if \(f\) is a function between sets, then input values correspond uniquely to output values. In other words, for every input value, there is only one ouptput value. Geometrically, this means that if you locate a domain value on the horizontal axis of the graph of \(f\text{,}\) then there is exactly one point on the vertical line through that domain value.
