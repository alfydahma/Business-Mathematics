  • Horizontal lines: Every point on a horizontal line has the same vertical coordinate regardless of the horizontal location. If we say this common \(y\) coordinate is \(b\text{,}\) then the horizontal line has equation \(y=b\text{.}\) Note that the slope of any horizontal line is \(m=0\text{.}\)
  • Vertical lines: Every point on a vertical line has the same horizontal coordinate regardless of the vertical location. If we say this common \(x\) coordinate is \(a\text{,}\) then the vertical line has equation \(x=a\text{.}\) Note that the slope of any vertical line is undefined.
