Solution 3.3.4.1.

From the previous example, we know that \(f'(x) = 4x-3\text{.}\) So, \(f'(0) = -3\) which means: When \(x=0\text{,}\) \(f(x)\) decreases at an approximate rate of 3 units per unit increase in \(x\text{.}\)
To find the tangent line, we need a point and a slope. Notice that we use \(f(x)\) fo find the point, and \(f'(x)\) fo find the slope.
  • A point is \((0,f(0)) = (0, 2(0^2) - 3(0)-1) = (0,-1)\)
  • slope = \(m = f'(0) = -3 \)
So, the tangent line is
\begin{align*} y-y_1 \amp = m(x-x_1) \\ y-(-1) \amp = -3(x-0) \\ y \amp = -3x -1 \end{align*}
