Definition 3.6.2.

Suppose \(f(x)\) is a function with the value \(c\) in its domain. Then \(f(x)\) has a relative (or local) maximum value of \(f(c)\) at \(x=c\) if \(f(c) \geq f(x)\) for every \(x\) near \(c\text{.}\) Similarly, \(f(x)\) has a relative (or local) minimum value of \(f(c)\) at \(x=c\) if \(f(c) \leq f(x)\) for every \(x\) near \(c\text{.}\)
