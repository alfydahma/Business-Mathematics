Example 1.2.1.

For this example we work with the line \(4x-2y=-4\) given in standard form (identify \(A,B,\) and \(C\) for homework). Let's find points on this line, \(L\text{,}\) by solving the equation for \(y\text{.}\) Verify for homework that \(4x-2y=-4\) is the same as \(y=2x+2\text{.}\) Now we can substitute values for \(x\) and compute \(y\text{:}\)
  • When \(x=1\text{,}\) we have \(y=2(1)+2=4\) which gives the point \((1,4)\) on \(L\text{.}\)
  • When \(x=0\text{,}\) we have \(y=2(0)+2=2\) which gives the point \((0,2)\) on \(L\text{.}\)
  • When \(x=-1\text{,}\) we have \(y=2(-1)+2=0\) which gives the point \((-1,0)\) on \(L\text{.}\)
All three of these points fall on the same line. Plot those points and graph this line for homework.
