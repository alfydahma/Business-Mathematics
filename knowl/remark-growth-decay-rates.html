Remark 5.2.1. Terminology and Determing Growth and Decay Rates.

With an exponential model of the form \(f(t) = y_0 b^{t}\text{,}\)
  • the value \(b-1\) is referred to as the (actual) growth rate if \(b > 1\)
  • the value \(1-b\) is the (actual) decay rate if \(0 < b < 1\text{.}\)
These values, of course, can be turned into percentages when multiplied by 100.
Assuming \(k > 0\text{,}\) with a model of the form \(f(t) = y_0 e^{kt}\) or \(f(t) = y_0 e^{-kt}\text{,}\)
  • the number \(k\) will be called the continuous growth rate if \(f(t) = y_0 e^{kt}\)
  • the number \(k\) will be called the continuous decay rate if \(f(t) = y_0 e^{-kt}\)
Again, this number an be turned into a percentage when multiplied by 100.
