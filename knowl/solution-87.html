Solution b.1.

First, the marginal profit function is \(P'(t) = -.3t^2 + 3.6t - 8.1\text{.}\) Now we solve the following for \(t\text{:}\)
\begin{align*} P'(t) \amp = 0\\ -.3t^2 + 3.6t - 8.1 \amp = 0 \quad \textrm{(multiply both sides by 10)} \\ -3t^2 + 36t - 81 \amp = 0 \\ -3(t^2 - 12t + 27) \amp = 0\\ -3(t-3)(t-9) \amp = 0 \end{align*}
and we find 2 criical numbers: \(t=3\text{,}\) which corresponds the year 2001 and \(t=9\text{,}\) which corresonds the year 2007.
