Each night you stay you get charged $159 plus an 11% tax. For \(n\) nights this means you'll pay \((159 + .11(159))n = 176.49n\text{.}\) With the $5 tip the total cost for \(n\) nights is
\begin{equation*} C=176.49n+5 \end{equation*}
In this context, we refer to \(n\) as the input variable and \(C\) as the output variable. So, the input variable \(n\) represents the number of nights stayed at the hotel. The output variable \(C\) represents the cost in dollars for staying at the hotel for \(n\) nights.
